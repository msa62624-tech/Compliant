## Additional Dashboards

The application also includes:

- **Broker Dashboard** - Accessible via `/broker-login` with broker credentials
- **GC Dashboard** - Accessible via `/gc-login` with general contractor credentials

These dashboards require authentication and specific user IDs to access properly. They feature similar layouts with role-specific functionality for managing insurance compliance within their respective workflows.
